Github + Hexo 并且通过Godaddy搭建静态个人网站详细教程（2）

更改主题

既然项目已经部署完毕了，网站也发布了，因为Hexo是一个很成熟的模版，所以也就是说，Hexo也提供了很多模版可以供替换。毕竟直男程序员可以忍受衣服和别的程序员撞衫，但是，网站主题，绝！对！不！可！以！
所以，少年，你想知道如何更换Hexo主题吗？

第一步：选择主题

Hexo官网提供了很多主题，可以从中挑选，pickHexo主题官网)，如果你不喜欢Hexo官方的主题，还可以选择第三方主题，或者去程序员大型交友网站Github去搜索下，或直接google。

第二步：安装主题

Hexo的文件目录中有两份配置文件（_config.yml），一份在根目录下，另一份在主题目录下，下文中将前者称为站点配置文件，后者为主题配置文件。
Hexo主题配置很简单，只需要将主题文件放置在根目录的themes目录下，然后修改配置文件即可。
我的网站使用的是NEXT主题，由于我没有图床，又不想自己的网站被拖慢，所以我尽量用NEXT官方图片来描述。

下载Next主题

  1. 前往 NexT 版本 发布页面。
  2. 选择你所需要的版本，下载 Download 区域下的 Source Code (zip) 到本地。例如，下载 v0.4.0 版本：
  3. 解压所下载的压缩包至站点的 themes 目录下， 并将 解压后的文件夹名称（hexo-theme-next-0.4.0）更改为 next.

    启动Next主题

    克隆/下载 完成后，打开 站点配置文件，找到 theme 字段，并将其值更改为 next

    验证主题是否启用

    在cmd界面输入 hexo s —debug，并且访问http://localhost:4000, 便可以在本地访问自己的网站。

第三步：配置

选择Scheme

Scheme 是 NexT 提供的一种特性，借助于 Scheme，NexT 为你提供多种不同的外观。同时，几乎所有的配置都可以 在 Scheme 之间共用。目前 NexT 支持三种 Scheme，他们是：

  • Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
  • Mist - Muse 的紧凑版本，整洁有序的单栏外观
  • Pisces - 双栏 Scheme，小家碧玉似的清新
    Scheme 的切换通过更改 主题配置文件，搜索 scheme 关键字。 你会看到有三行 scheme 的配置，将你需用启用的 scheme 前面注释 # 去除即可

选择语言

编辑 站点配置文件， 将 language 设置成你所需要的语言。建议明确设置你所需要的语言，例如选用简体中文，配置如下：
language: zh-Hans

语言 代码 设定示例
English en language: en
简体中文 zh-Hans language: zh-Hans
Français fr-FR language: fr-FR
Português pt language: pt or language: pt-BR
繁體中文 zh-hk 或者 zh-tw language: zh-hk
Русский язык ru language: ru
Deutsch de language: de
日本語 ja language: ja
Indonesian id language: id
Korean ko language: ko

菜单设置

菜单配置 主题配置文件的 meun， 若你的站点运行在子目录，请将连接前缀 / 去掉，默认支持的菜单项有：

键值 设定值 显示文本（简体中文）
home home: / 主页
archives archives: /archives 归档页
categories categories: /categories 分类页 （需要手动创建）
tags tags: /tags 标签页 （需要手动创建）
about about: /about 关于页面 （需要手动创建）
commonweal commonweal: /404.html 公益 404（需要手动创建）

菜单配置：
menu:
home: 首页
archives: 归档
categories: 分类
tags: 标签
about: 关于
search: 搜索
commonweal: 公益404

侧栏设置

默认情况下，侧栏仅在文章页面（拥有目录列表）时才显示，并放置于右侧位置。 可以通过修改 主题配置文件 中的 sidebar 字段来控制侧栏的行为。侧栏的设置包括两个部分，其一是侧栏的位置， 其二是侧栏显示的时机。

  1. 设置侧栏的位置，修改 sidebar.position 的值，支持的选项有：
  • left - 靠左放置
  • right - 靠右放置
  1. 设置侧栏显示的时机，修改 sidebar.display 的值，支持的选项有：
  • post - 默认行为，在文章页面（拥有目录列表）时显示
  • always - 在所有页面中都显示
  • hide - 在所有页面中都隐藏（可以手动展开）
  • remove - 完全移除
    sidebar:
    display: post

设置头像

编辑 主题配置文件， 修改字段 avatar， 值设置成头像的链接地址。其中，头像的链接地址可以是：

地址 值
完整的互联网 URI http://example.com/avatar.png
站点内的地址 将头像放置主题目录下的 source/uploads/ （新建 uploads 目录若不存在）配置为：avatar: /uploads/avatar.png 或者 放置在 source/images/ 目录下 配置为：avatar: /images/avatar.png

示例
avatar: http://example.com/avatar.png

设置作者名称

编辑 站点配置文件， 设置 author 为你的昵称。

站点描述

编辑 站点配置文件， 设置 description 字段为你的站点描述。站点描述可以是你喜欢的一句签名:)

其实，Hexo还可以使用第三方服务，比如访问服务或者百度统计。
这就未完待续了。
